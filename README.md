# Healthcare Microservices Project

Project ini adalah aplikasi layanan kesehatan berbasis backend services/microservice.

Struktur aktif project:

- `frontend`: aplikasi React, TypeScript, dan Vite.
- `services`: kumpulan backend service Ruby Sinatra.
- `observability`: fondasi OpenTelemetry.
- `docs`: dokumentasi roadmap, arsitektur, tech stack, dan panduan menjalankan project.
- `legacy`: versi lama, prototype, atau eksperimen yang belum menjadi struktur utama.

Dokumentasi utama ada di [docs/README.md](docs/README.md).
