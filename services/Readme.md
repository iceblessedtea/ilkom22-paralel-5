# Backend Services

Folder ini berisi backend services aktif untuk project layanan kesehatan.

Service yang tersedia:

- `patient-service`: data pasien.
- `doctor-service`: data dokter, ruangan, timeslot, dan jadwal.
- `appointment-service`: janji temu dan agregasi data pasien/dokter.
- `medical-record-service`: rekam medis.
- `api-gateway`: Nginx reverse proxy untuk mode Docker.

Panduan menjalankan service ada di:

- `README_RUN_OPTIONS.md`
- `../docs/RUNNING_GUIDE.md`
